# Animal Jam: ReJammed

## Installation
Currently, this project is only packaged for Windows and Linux 64-bit systems.

### Windows
- Go to the Releases page, and grab either the portable `.7z` or installer `.exe`
### Linux
#### Quick Install (Recommended)
- Run the following command in your terminal, ensure you have `jq` installed before running:
`bash -c "$(curl -fsSL https://raw.githubusercontent.com/thatoneham/Animal-jam-ReJammed/refs/heads/main/quick_install.sh)" -f`
- Updates can also be performed this way, as well as uninstalling the game.
#### Manual Install
- Linux users may want to install select versions of the game as opposed to the latest version.
- In this case, the appropriate `.tar.gz` portable release can be downloaded and installed via the included `install.sh` script.
#### Portable Mode
- As mentioned prior, portable releases can be simply downloaded and executed without any installation.
